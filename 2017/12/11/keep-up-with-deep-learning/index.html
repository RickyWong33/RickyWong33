follow一线人工智能/机器学习/深度学习/机器视觉最新动态的方法

发表于 2017-12-11 | | 阅读次数

AI / ML / DL / CV 每天日常爆炸式的发展中，学术创新和工业运用各种花样666，可以说是一天一个大新闻了。

客观来人工智能的核爆中心还是在英文世界，在和香港记者赛跑过程中，我觉得有必要把学习来源总结一把[1]。


Reddit

Reddit是一个全球性类似百度贴吧的社区，基于关键词主题形成了非常多的话题讨论。

机器学习和计算机视觉的许多成果都会被发帖在相应的Reddit社区中：

  • machine_learning
  • MachineLearning
  • computervision

arXiv

arXiv目前是由美国康奈尔大学运营的学术预印本发布的平台。非常多的学术工作者会习惯性的先把成果上传到arXiv上后再提交给专业的领域期刊。

一般来说，arXiv上的最新的文章还未进行深入的同行评审，虽然绝大多数也都会提交期刊接受评审，但是毕竟是一个较为早期的版本。但考虑到见刊评审的漫长时间，arXiv平台极大的推动了学术成果的快速交流，是一手获取学术发展的绝佳途径。

下面的栏目和我们关注的方向密切相关：

  • Artificial Intelligence
  • Learning
  • Neural and Evolutionary Computing
  • Computer Vision and Pattern Recognition

顺带安利一个配套arXiv的浏览器插件–Librarian–提供引文直链、BibTeX和评论交流的功能

arXiv-Sanity

arXiv-Sanity是基于arXiv的一个增强交互平台，重点囊括了人工智能方向上的研究。

ShortScience

ShortScience是一个学术交流平台。它的特点如其名，用最简单的话去概况科学著作。

GitXiv

GitXiv是arXiv的成果开源实现平台。我认为人工智能领域的快速发展离不开深度学习工具链（TensorFlow/Pytorch）和各大算法的开源化，大大降低了复现成本。GitXiv囊括了arXiv上所有具备开源实现的论文。

DataTau

DataTau是人工智能领域的Hacker News

Youtube频道

  • 3Blue1Brown
    是一个关于深度学习基础数学的频道，大量的可视化会让数学变得非常容易理解！
  • Two Minute Papers看名字你应该就很想关注，up主用最简短的语言概况最新的热点paper，非常值得推荐
  • Siraj Raval周更的一个浮夸up主= =

Quora

美版知乎，直接看Quora的好处是，有时候能看到一手的大佬解答

  • Machine Learning
  • Deep Learning
  • Computer Vision

  • Nvidia Deep Learning
