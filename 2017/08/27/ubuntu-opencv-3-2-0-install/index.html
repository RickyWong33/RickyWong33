ubuntu下编译安装OpenCV 3.2.0 + OpenCV_contrib

发表于 2017-08-27 | | 阅读次数

好久没有写文章了，这次介绍一下在Ubuntu上安装与配置OpenCV 3.2.0的过程。

OpenCV 3.2是2016年12月份发布的稳定版本；当然OpenCV的最新稳定版本已经到了3.3版本，是2017年8月份发布的。

从发布日志上，3.3版本大幅度的增加了深度学习的内容，所以如果你想体验更多与deep learning 相关的东西，你可以更新到最新版。但是从生产部署的稳定性来看，我们会优先采用有数个补丁的3.2版本，建议你也多等待一些3.3的版本关键修复后再升级。
